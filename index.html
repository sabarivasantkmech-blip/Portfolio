Privacy and secure access

Clear cookie policy and HTTPS-first deployment.

This portfolio is designed as a static site. It does not create a server-side user session by default, and it only stores small browser preferences unless the host adds analytics, contact forms, or login features later.

Cookie Policy

Essential preferences only

The site stores theme preference and cookie notice acknowledgement in the browser. No advertising cookies or analytics cookies are enabled in this template.

User Session Policy

No login session by default

If a future host adds accounts or forms, session cookies should be marked Secure, HttpOnly, SameSite=Lax or Strict, and should expire after a reasonable time.

HTTPS Certificates

Configured at the host

TLS certificates must be issued through the hosting provider or server, then paired with HTTPS redirects and HSTS headers for secure connections.
